# Portfolio Website

A modern, responsive portfolio website showcasing my work, experience, and projects. Built with cutting-edge web technologies and featuring real-time integrations with Spotify, Strava, and GitHub.

## Overview

This portfolio website serves as a comprehensive showcase of my skills, projects, and professional journey. It features a beautiful bento grid layout, interactive components, and seamless user experience.

## Technologies Used

- **Next.js 16** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **React 19** - Modern React features
- **Framer Motion** - Smooth animations
- **Radix UI** - Accessible component primitives
- **Magic UI** - Beautiful pre-built components

## Key Features

### Dynamic Content Integration

- **Spotify Integration** - Real-time display of currently playing or last played track
- **Strava Integration** - Latest activity tracking and statistics
- **GitHub Stats** - Live repository statistics and contributions

### Interactive Components

- **Reaction Counter** - Visitors can leave emoji reactions with visual particle effects
- **Tech Stack Cloud** - Interactive 3D icon cloud showcasing technologies
- **Bento Grid Layout** - Modular, responsive card-based design

### Content Management

- **Markdown-based Blog** - Easy-to-manage blog posts
- **Project Showcase** - Detailed project pages with markdown support
- **Work Experience Timeline** - Professional journey visualization

## Design Highlights

- Dark theme with custom color palette
- Responsive design for all screen sizes
- Smooth hover animations and transitions
- Accessible components with proper ARIA labels
- Optimized images and performance

## Development Process

This project was built with a focus on:

1. **Performance** - Optimized images, code splitting, and efficient rendering
2. **Accessibility** - WCAG compliant components and keyboard navigation
3. **User Experience** - Intuitive navigation and engaging interactions
4. **Maintainability** - Clean code structure and organized components
