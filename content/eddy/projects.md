# Edward's Projects

## CRYSTAL (April 2026)
An AI-powered institutional memory platform that ingests Slack and Google Drive communications, extracts decisions and entities via LLM, and surfaces them through a natural language query interface and interactive knowledge graph dashboard. Built on a dual-database architecture (Neo4j + pgvector) with an async ingestion pipeline and real-time Slack bot integration.

## Prospect Brief (April 2026)
An AI sales research agent that takes a company name, runs agentic web search via Claude, and outputs a structured prospect brief with AI readiness scores, pain points, and recommended talking points.

## Project Agila — Multi-Tenant Digital Document Approval System (August 2025 - Present)
Developed with a team of two: a multi-tenant digital document approval and review system with configurable multi-section workflows, role-based access control, and real-time notifications using Next.js, React, TypeScript, and Supabase. Features include a drag-and-drop form builder, workflow chain linking for multi-stage approvals, an auditor compliance view, and a real-time messaging system serving multiple organizations and business units.

## MISA's Event Registration System
A custom-built platform Edward helped develop to streamline operations for MISA (Ateneo Management Information Systems Association). The system reduced event registration time by 75% and now serves as the core platform for all MISA events. Features a centralized member database, event check-in system, real-time updates, voting eligibility tracking, and event history.

## Portfolio Website
Edward's personal portfolio at edwarddiesta.com, built with Next.js, TypeScript, React, and Tailwind CSS. Features a bento grid layout with live integrations — a Spotify card showing what he's listening to, a Strava card showing his latest run, and a GitHub stats card. Also includes a RAG-powered chatbot (Eddy), an interactive reaction counter with emoji particles, and markdown-based blog posts and project descriptions.

## SALBAR (Naga Mayoral Hackathon)
An offline-first emergency health protocol system built for when internet, power, and smart city systems fail. Created at the First Naga Mayoral Hackathon hosted by the Naga City Government and Mayor Leni Robredo.

## Hakot (Ship or Be Shipped Hackathon)
A platform that optimizes agricultural supply chains by utilizing empty trucks through AI-powered load matching for return trips. Built a cashless driver app using QRPH and PayRex to eliminate cash handling risks and manual remittance delays.

## FORTRESS (IM Summit IT Business Case Competition)
A FIDO2-based risk observation, evaluation, and security scoring system proposed at the IT Business Case competition hosted by MISA at IM Summit.
