# Edward's Projects

## MISA's Event Registration System
A custom-built platform Edward helped develop to streamline operations for MISA (Ateneo Management Information Systems Association), a 550-member organization. As one of two Lead Developers, he worked with a team of developers, designers, and project managers.

Key features include a centralized member database, event check-in system, real-time updates, voting eligibility tracking, and event history. The system has successfully powered 5 major events and counting. It replaced scattered spreadsheets with a single source of truth.

Technologies: Web development, database, UI/UX, systems design.

## Portfolio Website
Edward's personal portfolio at edwarddiesta.com, built with Next.js 16, TypeScript, React, and Tailwind CSS. It features a bento grid layout with live integrations — a Spotify card showing what he's listening to, a Strava card showing his latest run, and a GitHub stats card.

It also has an interactive reaction counter with emoji particles, markdown-based blog posts and project descriptions, and a clean, intentionally simple design philosophy. He chose to put everything on the home page so visitors don't miss anything.

## SALBAR (Naga Mayoral Hackathon)
An offline-first emergency health protocol system built for when internet, power, and smart city systems fail. Created at the First Naga Mayoral Hackathon hosted by the Naga City Government and Mayor Leni Robredo.

## Hakot (Ship or Be Shipped Hackathon)
A platform that optimizes agricultural supply chains by utilizing empty trucks through AI-powered load matching for return trips. Built a cashless driver app using QRPH and PayRex to eliminate cash handling risks and manual remittance delays.

## FORTRESS (IM Summit IT Business Case Competition)
A FIDO2-based risk observation, evaluation and security scoring system proposed at the IT Business Case competition hosted by MISA at IM Summit.
